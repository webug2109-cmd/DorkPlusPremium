## 🚀 Overview

**DorkPlusPremium** is a comprehensive, enterprise-grade security testing platform created by **Frostbyt3s**. It combines multiple security testing tools into one powerful, unified interface with advanced automation capabilities.

### ✨ Key Features

#### 🔍 **Core Scanning Modules**
- **Google Dork Generator** - 6 dork categories with 40+ templates
- **Web Crawler** - Asynchronous multi-threaded crawler with depth control
- **Keyword Extractor** - NLP-based security keyword extraction
- **SQLi Scanner** - 40+ payloads with DBMS detection (MySQL, PostgreSQL, MSSQL, Oracle, SQLite)
- **SQL Auto Dumper** - Complete database extraction including payment methods
- **Task Manager** - Real-time monitoring with filters and statistics

#### 🛠️ **Advanced Utilities**
- **Hash Identifier** - Identify MD5, SHA1, SHA256, SHA512, bcrypt, NTLM, MySQL5, Base64
- **Hash Generator** - Generate hashes with multiple algorithms
- **Base64 Encoder/Decoder** - Encode and decode Base64 strings
- **URL Encoder/Decoder** - URL encoding and decoding
- **User Agent Generator** - 10+ realistic user agents with rotation
- **Proxy Tester** - Bulk proxy validation with response time tracking
- **Port Scanner** - Scan 18 common ports

#### ⚙️ **License & Settings**
- **License System** - Keygen for 1day, 1week, 1month, 1year licenses
- **Hardware Binding** - Secure license-to-hardware binding
- **App Configuration** - User agent rotation, Cloudflare bypass, proxy settings
- **Bulk License Generation** - Generate up to 100 keys at once
