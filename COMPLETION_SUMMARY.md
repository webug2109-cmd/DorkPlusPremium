## 🚀 Getting Started

### For End Users

```bash
# Option 1: Quick Start (Linux/macOS)
git clone https://github.com/webug2109-cmd/DorkPlusPremium.git
cd DorkPlusPremium
bash install.sh dev

# Option 2: Docker (All platforms)

# Option 3: Windows
git clone https://github.com/webug2109-cmd/DorkPlusPremium.git
cd DorkPlusPremium
install.bat
run-dev.bat
```
